# ManageX


## Plan de Développement – Application de gestion de projets en temps réel



###  Phase 1 : Configuration du projet
- [ ] Création de la structure du projet  
- [ ] Mise en place de la base de données  

---

###  Phase 2 : Authentification et utilisateurs
- [ ] Création de la table `users`
- [ ] Interface de **connexion / inscription**
- [ ] Vérification et validation des champs
- [ ] Gestion de session et sécurité (hash des mots de passe)
- [ ] Gestion des rôles (Admin, Chef de projet, Membre, etc.)
- [ ] Mise à jour du profil utilisateur (photo, email, mot de passe)

---

###  Phase 3 : Gestion des projets
- [ ] Création / suppression / modification de projets 
- [ ] Détail des projets (description, deadline, priorité, statut) 
- [ ] Ajout de membres dans un projet  
- [ ] Affectation de tâches aux utilisateurs  
- [ ] Suivi d’avancement et Statistiques globales (progression, statut)
- [ ] Filtrage et recherche des tâches par statut, date, membre, etc.

---

###  Phase 4 : Collaboration en temps réel
- [ ] Système de messagerie instantanée  
- [ ] Indicateur de présence (“en ligne”, “hors ligne”)
- [ ] Notifications en direct
- [ ] Système de commentaires et mentions
- [ ] Synchronisation en direct des tableaux Kanban ou listes de tâches
- [ ] Mise à jour automatique de l’interface sans rechargement

---

###  Phase 5 : Interface et expérience utilisateur
- [ ] Design responsive et moderne  
- [ ] Thèmes clair/sombre  
- [ ] Animation et transitions fluides
